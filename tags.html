Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasis <em>technical sophistication</em> technical sophistication
strong strong highlight certain text <strong>technical sophistication</strong> technical sophistication
a hyperlink link text to web destinations <a href="wikipedia.org">wikipedia</a> wikipedia
img image embed an image into html <img src="images/small_twitter_logo.png">
code unicode displays text as unicode <code> some code <code> some code

Document Tags

Tag Name Purpose Example Result
html HyperText Markup Language define document type <html>(document) </html> a webpage
head Header Info Define elements of document <head>(code)</head> input title, meta, and other things
title Title Name of document <title>wikipedia<title> see "HTML Tags" on browser tag heading
meta Page Info Input key external facing elements <meta charset="utf-8"> Tells browser to use utf-8 character set
body Document Body Where the functional code lives <body> page code <body> text, photos, videos, everything
